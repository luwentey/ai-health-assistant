## 🎯 项目简介

AI Health Assistant 是一个**大模型驱动的个人健康管理工具集**，涵盖健康数据分析、饮食营养规划、运动方案推荐三大核心功能。项目展示了如何利用 AI 的推理能力处理个人健康场景中的多维度数据分析、个性化方案生成与风险评估。

### 核心能力

| 功能 | 描述 | AI 能力体现 |
|------|------|-------------|
| **健康分析** | 输入体检数据，输出健康评估报告 | 异常指标检测、趋势分析、风险预警 |
| **饮食规划** | 根据健康目标生成饮食方案 | 营养计算、热量平衡、禁忌识别 |
| **运动推荐** | 个性化运动计划生成 | 体能评估、渐进式训练、安全建议 |

## 🤖 AI 驱动能力展示

### 长链推理示例

```
用户输入：男，35岁，身高175cm，体重85kg，体脂率28%，总胆固醇5.8mmol/L
  ↓
AI 分析链：
  BMI = 85/(1.75)² = 27.8 → 超重（正常18.5-24）
  → 体脂率28% > 25%(男) → 肥胖型超重
  → 胆固醇5.8 > 5.2(正常上限) → 高胆固醇血症
  → 综合诊断：代谢综合征早期
  ↓
方案推导：
  热量摄入 ← 基础代谢 = 66+13.7×85+5×175-6.8×35 ≈ 1780kcal
  → 减脂热量 = 1780×0.8 ≈ 1420kcal
  → 蛋白质需求 = 85×1.8 ≈ 153g
  → 运动建议：低强度有氧(降胆固醇) + 力量训练(增肌减脂)
  → 禁忌：高饱和脂肪、高糖、长期空腹有氧
```
