# AI Health Assistant

> English Version | [中文](../README.md)

AI-Driven Personal Health Management Toolkit — Demonstrating LLM capabilities in multi-dimensional health data analysis, personalized nutrition planning, and exercise prescription.

## Core Features

| Module | Function | AI Capability |
|--------|----------|---------------|
| **Health Analyzer** | Input medical data → output health report | Anomaly detection, trend analysis, risk assessment |
| **Diet Planner** | Generate meal plans by health goals | Calorie calculation, nutrition balance, allergy screening |
| **Exercise Recommender** | Personalized workout plan | Fitness assessment, progressive training, safety |

## Quick Start

```bash
# Health analysis
python scripts/health_analyzer.py

# Diet planning
python scripts/diet_planner.py

# Exercise recommendation
python scripts/exercise_recommender.py
```
