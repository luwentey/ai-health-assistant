## 2. 模块设计

### 2.1 health_analyzer — 健康分析引擎

```
输入：性别/年龄/身高/体重/体脂率/血常规指标
  ↓
阶段1：身体指标计算
  ├── BMI = 体重(kg) / 身高(m)²
  ├── 体表面积 = 0.0061×身高 + 0.0128×体重 - 0.1529
  └── 基础代谢率(BMR) = 计算值
  ↓
阶段2：标准对比
  ├── BMI 分级（偏瘦/正常/超重/肥胖）
  ├── 体脂率分级
  ├── 血脂/血糖标准对比
  └── 综合评分
  ↓
阶段3：AI 推理
  ├── 异常指标关联分析
  ├── 代谢综合征风险评估
  ├── 心血管风险等级
  └── 针对性改善建议
  ↓
输出：结构化健康报告
```

### 2.2 diet_planner — 饮食规划模块

```
输入：健康数据 + 目标（减脂/增肌/维持）+ 偏好
  ↓
步骤1：热量计算
  ├── 维持热量 = BMR × 活动系数
  └── 目标热量 = 维持热量 ± 调整值
  ↓
步骤2：宏量营养分配
  ├── 蛋白质 = 体重 × 系数
  ├── 脂肪 = 总热量 × 20-30%
  └── 碳水 = 剩余热量
  ↓
步骤3：食谱生成
  ├── 三餐分配（3:4:3）
  ├── 食物选择（蛋白质来源/碳水来源）
  ├── 禁忌检查（疾病限制）
  └── 微量元素估算
  ↓
输出：每日饮食方案
```

### 2.3 exercise_recommender — 运动推荐模块

```
输入：健康数据 + 运动史 + 目标
  ↓
评估：
  ├── 心血管健康等级
  ├── 肌肉骨骼状况
  ├── 运动经验水平
  └── 时间约束
  ↓
生成：
  ├── 有氧运动（频率/强度/时长）
  ├── 力量训练（动作/组数/次数）
  ├── 渐进计划（周为单位递增）
  └── 安全提示（禁忌/警告）
  ↓
输出：个性化运动方案
```

## 4. 健康标准库

### BMI 标准 (WHO)

| 分类 | BMI 范围 |
|------|:--------:|
| 偏瘦 | < 18.5 |
| 正常 | 18.5 - 24 |
| 超重 | 24 - 28 |
| 肥胖 | ≥ 28 |

### 体脂率标准

| 性别 | 正常范围 | 肥胖标准 |
|:----:|:--------:|:--------:|
| 男 | 15-20% | > 25% |
| 女 | 20-25% | > 30% |
