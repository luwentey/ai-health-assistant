# AI Health Assistant — 需求分析

> 项目：AI 驱动的个人健康管理工具 | 日期：2026-05-14

---

## 1. 项目目标

构建一个 AI 辅助的个人健康管理工具，利用大模型的推理能力提供个性化的健康分析、饮食规划和运动建议。

## 2. 目标用户

| 用户画像 | 需求描述 |
|----------|----------|
| 办公室白领 | 久坐、不规律饮食、缺乏运动 |
| 健身爱好者 | 需要科学饮食配合训练计划 |
| 三高人群 | 需要饮食禁忌提醒和健康监测 |
| 减脂人群 | 需要热量控制和运动规划 |

## 3. 功能需求

### 3.1 健康分析 (P0)

| 需求 | 描述 | AI 贡献 |
|------|------|---------|
| 体检数据输入 | 身高/体重/体脂率/血常规等 | 自然语言解析非结构化数据 |
| 健康指标计算 | BMI、体脂率分级、基础代谢 | 自动运算 + 标准对比 |
| 异常检测 | 识别超标的健康指标 | 多指标交叉分析 |
| 健康评分 | 综合健康指数评分 | 加权评分模型 |
| 风险预警 | 代谢综合征、心血管风险等 | 多指标联合分析 |
| 趋势分析 | 历史数据变化趋势 | 时间序列分析 |

### 3.2 饮食规划 (P0)

| 需求 | 描述 | AI 贡献 |
|------|------|---------|
| 热量计算 | 基础代谢 + 活动消耗 | 个体化计算模型 |
| 营养配比 | 碳水/蛋白质/脂肪比例 | 目标导向的宏量营养分配 |
| 食谱生成 | 每日/每周饮食方案 | 考虑饮食偏好和禁忌 |
| 禁忌筛查 | 排除不适合的食物 | 疾病知识图谱匹配 |
| 营养追踪 | 摄入营养数据分析 | 实时评估与调整建议 |

### 3.3 运动推荐 (P1)

| 需求 | 描述 | AI 贡献 |
|------|------|---------|
| 体能评估 | 基于用户数据的体能水平评估 | 多维度综合评价 |
| 运动方案 | 个性化的运动计划 | 渐进式训练设计 |
| 安全建议 | 运动禁忌和风险提示 | 健康状况关联分析 |
| 进度追踪 | 运动完成度评估 | 阶段性调整建议 |

## 4. 非功能需求

| 需求 | 标准 |
|------|------|
| 响应时间 | 单次分析 ≤ 3s |
| 准确率 | BMI/基础代谢计算误差 < 1% |
| 可用性 | CLI 工具，无需图形界面 |
| 扩展性 | 支持添加更多健康指标和分析规则 |
| 隐私性 | 所有数据本地处理，不联网 |
